# flappy-bird-bot-using-reinforcement-learning
Flappy bird agent in pytorch(python) that learns to play the game 
using Deep Q learning algorithm developed by google deepmind.

# Installation Dependencies:
* 'Python3.6'
* 'Pytorch - 1.0'
* 'OpenCV-Python - 3.4.4'
* 'Pygame - 1.9.4'
* 'Numpy - 1.15.4'
